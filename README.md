# Excel-Sales-Dashboard-Analysis
Interactive Excel Sales Dashboard analyzing regional sales performance, target achievement, and executive-wise insights using Pivot Tables and Charts.
# Excel Dashboard Project

This Excel project demonstrates data cleaning, analysis, and visualization using Excel.  

## Project Highlights
- *Data Cleaning:* Removed duplicates, handled missing values, and formatted data for analysis.  
- *Pivot Tables:* Summarized large datasets to extract key insights.  
- *Formulas:* Used Excel formulas to calculate totals, percentages, and metrics.  
- *Dashboard:* Created an interactive dashboard to visualize trends and performance.
